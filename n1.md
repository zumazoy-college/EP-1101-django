# Работа №1 «Создание Django-проекта»

## Краткая справка по командам и настройкам

### 1. Создание проекта и виртуального окружения

```bash
# Создание папки проекта и переход в неё
mkdir EP-1101-django && cd EP-1101-django

# Инициализация проекта с виртуальным окружением (uv)
uv init

# Установка Django
uv add django
```

### 2. Создание Django-проекта и приложения

```bash
# Создание проекта с именем "zuma" (точка — в текущей папке)
uv run django-admin startproject zuma .

# Создание приложения "ep1101"
django startapp ep1101
```

#### Для удобства была создана PowerShell-функция:

```bash
function django { uv run python manage.py $args }
```

#### Теперь команда `django` заменяет `uv run python manage.py`

## Изменения в файлах

### 3. Настройка `zuma/settings.py`

#### Добавление приложения в `INSTALLED_APPS`:

```py
INSTALLED_APPS = [
    'ep1101',  # ← добавлено
]
```

#### Настройка директории для шаблонов:

```py
TEMPLATES = [
    {
        'BACKEND': 'django.template.backends.django.DjangoTemplates',
        'DIRS': [BASE_DIR / 'templates'],
        'APP_DIRS': True,
        # ... остальные настройки
    },
]
```

### 4. Создание папки для шаблонов

```bash
mkdir templates
```

### 8. Запуск проекта

```bash
# Применение миграций базы данных
django migrate

# Создание суперпользователя (админа)
django createsuperuser

# Запуск сервера разработки
django runserver
```
